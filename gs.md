## A指与个股之间的关系



| 买入                          | 卖出            | 效果   |
| :-------------------------- | ------------- | ---- |
| A指昨日收阴今日收阳                  | 个股明日后再次收阴     |      |
| A指昨日收阴今日收阳，个股昨日收阴，今日不管收阴还是阳 | 个股明日后再次收阴     |      |
| A指十日均线转弯向上，个股十日均线比A指晚       | 个股十日均线向下，比A指早 |      |
|                             |               |      |
